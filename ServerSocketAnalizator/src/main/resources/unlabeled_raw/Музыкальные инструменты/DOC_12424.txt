Обзор гитары:/
 /
Первая 8-струнная электрогитара Cort с переменной мензурой, KX508MS оснащена высоко оцененными звукоснимателями Fishman Fluence Modern которые обеспечивают разнообразное отличное звучание для современного рок-музыканта. /
Комплект хамбакеров Fishman Fluence Modern./
 Нажимно-отжимные переключатели облегчают отключение катушек для создания звучания синглов, а также Voice1 от Fishman для современного активного тона хамбакера с высокой выходной мощностью и Voice2 для пассивного типа с органичными ощущениями с повышенной винтажной динамикой и откликом, сохраняя при этом высокую выходную мощность для теплых и насыщенных искажений или чистого звучания./
Индивидуальный бридж каждой струны./
 Индивидуальный фиксированный бридж для каждой струны, проходящей сквозь корпус, позволяет значительно улучшить сустейн и разделить струны для большей артикуляции и точности интонации./
