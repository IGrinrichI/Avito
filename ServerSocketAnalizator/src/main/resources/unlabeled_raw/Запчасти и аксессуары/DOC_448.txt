БЕСПЛАТНАЯ ЗАМЕНА МАСЛА ДЛЯ НОВОГО ВЛАДЕЛЬЦА BMW С ПРОБЕГОМ В ОФИЦИАЛЬНОМ ДИЛЕРСКОМ ЦЕНТРЕ BMW. Акция проводится до 31 января 2020 года. В период действия кампании, владельцы автомобилей BMW с пробегом, приобретенных на вторичном рынке, получают услугу по замене моторного масла в официальном дилерском центре BMW в течении 60 дней со дня государственной регистрации автомобиля (постановки на учет). — Дата регистрации отсчитывается от даты регистрации, указанной в свидетельстве о регистрации транспортного средства. — Услуга по замене моторного масла включает в себя: необходимое для замены количество моторного масла из линейки BMW спецификации LL01/LL04 5W30, масляный фильтр и работы по замене (кроме автомобилей BMW M). — Услуга по замене моторного масла для автомобилей BMW M включает в себя: необходимое для замены количество моторного масла из линейки BMW, согласно техническим рекомендациям для автомобилей BMW M, масляный фильтр и работы по замене. — Предложение действительно только для физических лиц. — Предложением можно воспользоваться только один раз в период действия кампании.
