Поплавки катамарана Продается:поплавки катамарана(алюминий),предназначены для обеспечения непотопляемости конструкции.Могут быть использован для изготовления плавательного комплекса для купания. Поплавки разделены на 4 больших отсека и 5 маленьких,закрываемых винтовыми пробками.Внутри отсеков имеются продольные ребра жесткости.На отсеки устанавливаются по 2 профилированные герметичные стойки с прямоугольными фланцами,для крепления верхней рамной платформы.На платформе устанавливаются необходимые механизмы,можно оборудовать каюты.Катамаран может быть использован при подъеме каких-либо грузов со дна моря,как развлекательный комплекс,для оборудования жилых кают.Торг уместен.Возможна продажа по безналичному расчету
