Ethernet Shield W5100 для Arduino / Ардуино Плата расширения Arduino Ethernet Shield позволяет плате Arduino подключаться к интернету. Она построена на базе ethernet микросхемы Wiznet W5100. Wiznet W5100 обеспечивает сетевой (IP) стек, способный работать и с TCP, и с UDP. Она поддерживает до четырех одновременных сокетных соединений. Для написания скетчей используйте библиотеку Ethernet, которая позволяет подключаться к интернету с помощью платы Ethernet Shield. Ethernet плата подключается к плате Arduino с помощью длинных разъемов, которые проходят сквозь плату расширения для дальнейшего использования. Это позволяет сохранить распиновку неизменной и использовать другие платы расширения, подключая их сверху.
