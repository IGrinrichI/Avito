Вспышка Yongnuo YN685C для Canon Вспышка Yongnuo YN685C для Canon. Новая. В упаковке./
/
Вспышка сама по себе имеет совершенно новый дизайн корпуса и широкий набор функций, пригодных для старшей модели Speedlite, включая TTL, HSS и поддержку внешних батарей. Можно использовать либо одну YN685 на камеру для управления другими вспышками, или можно выбрать YN622C-TX или YN560-TX для управления всей внешней осветительной аппаратурой./
/
