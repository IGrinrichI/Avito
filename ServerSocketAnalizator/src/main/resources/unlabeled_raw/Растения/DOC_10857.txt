Бонсай кармона (Carmona microphylla) - вечнозеленый бонсай, сформированный из кустарникового растения Кармона мелколистная. Имеет второе название "Чайное Дерево", т.к. листья Кармоны используют вместо чая, а также для лечения кашля. Ствол S-образной формы, хорошо разрастается. Для придания формы необходима обрезка побегов для сохранения миниатюрной формы. Цветет круглогодично белыми мелкими цветочками. По Фен Шуй Бонсай Кармона считается благоприятным для домашнего содержания растением, т.к. несет положительную энергетику. Является неприхотливым Бонсай. /
