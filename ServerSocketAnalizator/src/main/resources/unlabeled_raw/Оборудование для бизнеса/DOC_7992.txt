Станок для фиксации крс Ветмастер Волгоградский Завод Весоизмерительной Техники производит станки для фиксации КРС Ветмастер./
 /
Ветеренарная станция позволяет проводить измерение веса животного, ветеринарную и зоотехническую обработку (обработку копыт и рогов КРС) с фиксацией животного. Она способна решать любые задачи в области ветеринарного обслуживания с одновременным измерением веса животных./
 /
Основные элементы /
1. Рычаги управления, не требующие больших усилий/
/
2. Ограничитель бокового движения (Боковой зажим)/
/
3. Ограничитель продольного движения/
/
4. Фиксатор головы животного/
/
5. Сквозные двери для обслуживающего персонала/
/
6. Автоматически запирающаяся задняя дверь-слайдер/
/
7. Механизм торможения фиксатора головы животного (Надежный механизм, не требующий специального обслуживания)/
/
8. Дополнительный рычаг управления, позволяющий одному оператору осуществить все необходимые действия по фиксации животного/
/
9. Открывающиеся боковые стенки/
/
10. Дополнительное окно для осуществления ветеринарных процедур и обработки копыт животных./
